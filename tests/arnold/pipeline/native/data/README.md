# Golden Graph Traces for Native/Graph Parity

Generated by ``_gen_golden_traces.py`` using the existing
``parity_trace.py`` normalization helpers.

## Files

| File | Case | Description |
|------|------|-------------|
| ``full_run.json`` | full-run | Complete toy pipeline execution through graph executor |
| ``resume.json`` | resume | Full graph run (reference for forced-resume parity) |

## Coverage

| M3 Case | Coverage | Note |
|---------|----------|------|
| Control override | ❌ | Requires T14 ``toy_megaplan.py`` fixtures |
| Additive override | ✅ partial | Typed producer/consumer flow in full_run |
| Guarded loop | ✅ | ``should_loop``/``body`` loop in full_run |
| Nested subpipeline promotion | ❌ | Requires T14 fixtures |
| Suspension/resume composite cursor | ❌ | Requires T14 fixtures |

## Determinism

All traces are deterministic: timestamps, run IDs, artifact-root paths,
and sequence numbers are masked by the normalization helpers.
Re-running ``_gen_golden_traces.py`` from a clean counter state
produces identical output.
