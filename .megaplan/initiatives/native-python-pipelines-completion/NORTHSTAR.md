# Native Python Pipelines Completion North Star

Arnold runs entirely on native Python pipelines. Every package builds native declarations and returns projected `Pipeline` compatibility shells through `Pipeline.native_program`. Execution, resume, and testing use native runtime semantics and native traces as runtime/dispatch/resume evidence. Graph-era surfaces are removed or explicitly shimmed only after import inventory proves it is safe.

`Pipeline.native_program` is the dispatch and compatibility substrate for this
epic, not the final Megaplan authoring truth. This epic clears the old graph
runtime out of the way so the composition follow-up can make Megaplan product
semantics visible in canonical workflow source, declared policy, or audited pure
phase bodies. A non-null `native_program`, native trace, route label, or
projected shell is substrate evidence only unless the report-owned semantics
are visible at that higher level.

## Required Anchor

This epic is governed by `docs/arnold/megaplan-native-representation-report.md`.
That report is the durable Megaplan end-state target: product semantics must
eventually be visible as native Python workflow structure, not hidden inside
opaque handlers.

`docs/arnold/megaplan-native-representation-alignment-plan.md` is the required
enforcement plan. Milestone briefs must cite the matrix rows they affect and
must say whether they provide substrate proof or report conformance.

This completion epic is responsible for preserving or establishing the substrate
needed to reach that target: native execution, resume, trace, package, and test
truth must not block later visible-native Megaplan composition. It does not have
to deliver the full report shape by itself, but it must not make any report
requirement harder to satisfy without recording the deferral and owner.
Native-backed canonical Megaplan, non-null `native_program`, route labels, and
native-owned goldens are substrate proof only unless Megaplan product semantics
are actually visible in canonical workflow source, declared policy, or an
audited pure phase body.
