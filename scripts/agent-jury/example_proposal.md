# EXAMPLE PROPOSAL — For Testing the Agent Jury

```
PROPOSAL:
Entity A is considering deploying an AI-powered surveillance system across all of its
public-facing facilities (approximately 200 locations across Region B). The system would:

1. Continuously monitor all public areas via camera feeds
2. Run real-time facial recognition against a database of known security threats
3. Flag "anomalous behavior" using behavioral analysis ML models
4. Automatically alert human security personnel for review
5. Retain all footage for 90 days; flagged footage retained indefinitely

Stated benefits:
- Expected 60% reduction in security incidents based on pilot at 5 locations
- Faster response to threats (estimated 4-minute reduction in response time)
- Cost savings of approximately X units/year through reduced security staffing
- Deterrent effect on potential bad actors

Stated costs:
- Initial deployment: Y units (one-time)
- Annual operation: Z units/year
- Privacy impact on visitors and employees
- Potential for bias in facial recognition systems (vendor reports 2-4% higher false
  positive rates for certain demographic groups)

Decision to be made:
Whether to approve full deployment, extend the pilot, require modifications, or reject.

Alternative: Extend pilot to 20 locations for 12 more months with independent audit.

REQUESTED_LENSES: risk, ethics, utility, systems, devil
URGENCY: high
```
