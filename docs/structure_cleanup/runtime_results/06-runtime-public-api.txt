### 5. Flat Duplicates of `eval/` Package — Replace with Shims

These four flat files at `runtime/*.py` have near-identical counterparts inside `runtime/eval/`. The canonical content now lives in the `eval/` package.

#### `runtime/eval.py` (437 lines)
- **Completely shadowed** by `runtime/eval/__init__.py` package. Python never loads this module.
- `from vibecomfy.runtime.eval import compile_eval_subgraph` resolves to `runtime/eval/__init__.py` → `eval/core.py`.

**Action:** `DELETE` — zero risk

#### `runtime/eval_plan.py` (187 lines)
- Duplicate of `eval/plan.py`.
- External importer: `tests/test_agentic_affordances.py:25` (`from vibecomfy.runtime.eval_plan import plan_eval_node`).
- Internal importer: `runtime/eval_prompt.py:11` (itself a deletion candidate).

**Action:** Replace body with `from vibecomfy.runtime.eval.plan import *  # noqa: F401, F403` — one-line shim keeps the existing import path alive.

#### `runtime/eval_prompt.py` (146 lines)
- Duplicate of `eval/prompt.py`.
- External importer: `tests/test_agentic_affordances.py:12` (`import vibecomfy.runtime.eval_prompt as eval_prompt_module`). **Monkeypatches** `comfy_server`, `ComfyClient`, `_wait_for_server_history` on the module, then calls `eval_node`.

**Action:** Replace with shim `from vibecomfy.runtime.eval.prompt import *  # noqa: F401, F403` **AND** update the test to patch `vibecomfy.runtime.eval.prompt` instead of `vibecomfy.runtime.eval_prompt`. Without the test fix, monkeypatches silently no-op on a star-import shim.

#### `runtime/preview_types.py` (83 lines)
- Duplicate of `eval/preview_types.py`.
- Only importer: `runtime/eval_plan.py:8` (itself becoming a shim).

**Action:** Replace body with `from vibecomfy.runtime.eval.preview_types import *  # noqa: F401, F403`

### 6. Accidental Flat Namespace Clutter (Minor)

#### `runtime/__init__.py` line 1: `from .run import run, run_embedded, run_embedded_sync, run_sync, smoke_runtime, smoke_runtime_sync`

`smoke_runtime` and `smoke_runtime_sync` are exported by `runtime/__init__.py` but **not** by `vibecomfy.__init__`. They're visible as `vibecomfy.runtime.smoke_runtime`. No external consumer found.
