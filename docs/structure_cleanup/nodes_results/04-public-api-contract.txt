## 4. Best Cleanup That Improves Structure

Already planned in `revised_plan.md`: **Replace the 19 hardcoded wildcard re-exports in `vibecomfy/nodes/__init__.py`** with a programmatic loop:

```python
# vibecomfy/nodes/__init__.py
from vibecomfy.nodes._generated import MODULES

__all__: list[str] = []
for mod_name in MODULES:
    mod = __import__(f"vibecomfy.nodes._generated.{mod_name}", fromlist=[mod_name])
    names = getattr(mod, "__all__", [])
    for name in names:
        globals()[name] = getattr(mod, name)
    __all__.extend(names)
```

The individual shim files (`nodes/core.py`, `nodes/kjnodes.py`, etc.) **stay** — they are the stable public API surface. The `__init__.py` barrel gets DRY'd up. The `_generated/__init__.py` already exports `MODULES` for exactly this purpose.
