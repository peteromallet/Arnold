The per-pack shims (e.g. `core.py`) are trivial 4-line re-exports:

```python
from vibecomfy.nodes._generated import core as _generated
from vibecomfy.nodes._generated.core import *
__all__ = list(_generated.__all__)
```

These exist solely so that `from vibecomfy.nodes.core import X` works — no internal code actually imports them that way except `nodes/__init__.py` itself. The emitter (`emit_constants.py`) imports directly from `vibecomfy.nodes._generated.{module}`.

### 1. Target Layout

```
vibecomfy/nodes/
├── __init__.py       ← star-imports from _generated modules (NOT from sibling shims)
├── __init__.pyi      ← same
├── index.py          ← kept as-is (not generated)

    ├── __init__.py   ← MODULES list (generated)
```

**`vibecomfy/nodes/_generated/` is the sole home for all per-pack generated code.**
