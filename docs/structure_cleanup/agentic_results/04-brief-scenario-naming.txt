## 3. Is Renaming Worth Doing?

**Yes — low risk, high payoff.** The internal `name:` field already uses hyphens consistently, so no YAML content needs changing. The bug is purely in the filename. Renaming the 9 files to match the hyphen convention would:

- Make `ls` / glob matches intuitive again
- Eliminate the "why didn't my grep find it?" class of bugs
- Make tooling that derives scenario paths from brief slugs work correctly

No other files reference these filenames by underscore (all cross-references go through the `name:` field), so there's zero cascade impact. Renaming is a 30-second operation.
