### Risk Detail: `this.env`

This file contains **live API keys** for OpenAI, Anthropic (Claude), Gemini, and Fal.ai. It is:
- Listed in `.gitignore:36` — so currently protected from accidental `git add`.
- **Not tracked** by git (confirmed by `.gitignore` entry).
- Referenced in 0 source files, 0 tests, 0 scripts — purely a local development convenience file.
- Already flagged in `docs/megaplan_chains/pristine_cleanup/audit/10-repo-hygiene.md` as a security risk.

**Recommendation:** Replace with a `.env.example` template (showing keys with placeholder values) and move the real file to `~/.config/vibecomfy/env` or the user's shell profile. Then delete `this.env` locally. **Do not delete until the replacement is in place** — the file is currently the developer's only API key store.
