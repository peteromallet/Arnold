# Tests Layer Audit 05: Runtime And Smoke Tests

Audit runtime, smoke, RunPod, e2e, and server/embedded test tiers.

Questions:
- Which tests require external services, GPU, ComfyUI, RunPod, or network?
- Are markers and directory names aligned?
- Are generated outputs or reports committed or just ignored locally?
- Should README docs clarify how to run these tiers?

Do not suggest deleting expensive tests unless they are duplicated or stale.
