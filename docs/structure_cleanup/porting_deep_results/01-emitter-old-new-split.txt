**Migration steps:**

2. Run full test suite
3. Delete `ui_emitter.py`

#### `emitter.py` → `emit/emitter.py` migration: **LOW-MEDIUM RISK**
- All 6 callers import names that exist in the new `emit/emitter.py` (either directly defined or re-exported from split modules).

**Migration steps:**
