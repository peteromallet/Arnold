## 🔍 Recommended Stale-Path Scan After Edits

After any moves, run these scans to catch missed references:

### 1. Catch absolute `docs/*.md` stale paths
```bash
rg -n 'docs/[a-zA-Z_/-]+\.md' . --type md --type py --type sh 2>/dev/null | grep -v 'docs/structure_cleanup/' | grep -v 'node_modules/'
```

### 2. Catch bare-relative markdown links that break
```bash
rg -n '\([a-zA-Z_/-]+\.md\)' . --type md 2>/dev/null | grep -v 'docs/structure_cleanup/' | grep -v 'node_modules/'
```

### 3. Catch code-file string references
```bash
rg -n '"docs/[a-zA-Z_/-]+\.md"' . --type py --type rs 2>/dev/null
```

### 4. Special: `vendor/ComfyUI/` paths
`vendor/ComfyUI/README.md:125` links to `docs/custom_nodes.md`. Moving that file breaks a link in a vendored copy. Check if that's a symlink or a real copy.
