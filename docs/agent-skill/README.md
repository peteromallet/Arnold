# Agent Skill

This directory is the single authored source for the VibeComfy agent skill.

- Edit `SKILL.md` here.
- Keep root `AGENTS.md` and `CLAUDE.md` as short bootstraps only.
- Run `python scripts/sync_agent_skill.py --apply` after editing the skill.
- Run `python scripts/sync_agent_skill.py --install-user` to symlink this
  directory into local Claude, Codex, and Hermes skill directories.
