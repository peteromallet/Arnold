**Answer:** It is fully plumbed, not dead metadata. `executor.py` has `_find_native_program(pipeline)` (line 286–297) which checks `pipeline.native_program` first, then falls back to resource bundles. Verified** against `executor.py:166-212, 286-297`, `native/routing.py:100-161`, `flags.py:1-22`.
