# Archive

This directory holds historical design notes, migration runbooks, archived YAML chain specs, generated plans, sprint briefs, and one-off review artifacts that are useful for context but are no longer current operator guidance.

Current docs should link here only when they need historical background. New setup, deployment, and workflow instructions belong in the live docs at `docs/`.

## Contents

- `sprints/` - historical sprint ideas, handoffs, cleanup inventories, and sprint briefs.
- `megaplan/` - dated notes from specific Megaplan runs or operator friction reports.
- Root-level files - migration plans, old chain specs, and archived design plans that predate the current docs layout.
